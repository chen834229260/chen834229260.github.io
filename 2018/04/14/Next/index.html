Next主題的安裝以及优化

发表于 2018-04-14 | 分类于 Hexo主题 |

前言

记录一下Next主题的安装和优化,教程全部来源于百度,自己简单做了个总结.

安装主题

安装的过程就一行代码,需要在博客的根目录打开git Bash Here,输入如下命令:

git clone https://github.com/theme-next/hexo-theme-next themes / next
如下载完成.就可以在博客根目录下的thmees中找到一个为next的文件夹

使用主题

当主题下载完成之后,只需要去博客根目录下的配置文件_config.yml中搜索theme,将其修改为主题的名字

在博客根目录下的Git Bash Here中输入:

hexo d -g

提示上传成功的话,就可以在自己的博客网站看到Next的默认主题效果了

Next主题的设置

当然,Next主题为我们提供了三种不同的主题样式,我们可以通过修改主题的配置文件来对主题的默认样式进行修改,以此找到自己喜欢的样式,themes/next/_config.yml;
