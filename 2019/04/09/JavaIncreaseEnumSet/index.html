Java提高集合之EnumSet

发表于 2019-04-09 | 分类于 JAVA |

前言

　接触JAVA一年有余,对于JAVA基础从未认真的进行过总结,导致一些基础问题云里雾里,故开篇对JAVA基础做一次总结,本篇为第八篇,JAVA集合中的EnumSet;

正篇

EnumSet的概念

　EnumSet是Java枚举类型的泛型容器,EnumSet中的所有元素都必须是指定枚举类型的枚举值

EnumSet的特性

　1. EnumSet的集合元素也是有序的,EnumSet以枚举值在Enum类的定义顺序来决定集合元素的顺序
　2. EnumSet在内部以位向量的形式存储,这种存储形式非常紧凑,高效,因此EnumSet对象占用内存很小,而且运行效率很好,尤其是进行批量操作时,如果其参数也是EnumSet集合,则该批量操作的执行速度也非常快
　3. EnumSet集合不允许加入null元素,如果视图插入null元素,EnumSet将抛出NullPointerException异常
　4. EnumSet类没有暴露任何构造器来创建该类的实例,程序应该通过它提供的类来创建EnumSet对象
　5. 如果只是想删除EnumSet是否包含null元素或试图删除null元素都不会抛出异常,只是删除操作将返回false,因为没有任何null元素被删除
